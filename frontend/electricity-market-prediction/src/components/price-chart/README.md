# Price Chart Feature

Core feature for visualizing electricity price data and predictions.

## Structure
- `components/`: UI components (PriceChart, controls, secondary charts).
- `context/`: State management (PriceChartContext).
- `hooks/`: Custom hooks for data fetching and processing.
- `plugins/`: Lightweight Charts plugins and extensions.
- `types/`: Type definitions and interfaces.
- `utils/`: Helper functions specific to price charts.
- `constants.ts`: Configuration constants.
- `index.ts`: Public API exports.

## Key Components
- `PriceChart`: Main container component.
- `PriceChartLightweight`: Implementation using Lightweight Charts library.
